This is LCC demostration application for Graphics Library 4.00.


SUPPORTED HARDWARE STARTER KITS
-------------------------------

PIC32MZ Embedded Connectivity (EC) Starter Kit

PIC32MX USB Starter Kit II


KNOWN ISSUES AND WORK-AROUNDS
-------------------------------

For PIC32MZ EC, after programming, the application may not display the UI folders as expected. If this is 
the case, drag your finger across the touch display from left to right to activate the UI display.
